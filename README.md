# TangoSNS
![TangoSNS](./img/logo.png "ロゴ")
TangoSNSは、自分だけの単語帳を作成し、他のユーザーと共有・学習できるソーシャル単語帳プラットフォームです。

## アプリケーションの概要
ユーザーは独自の単語帳を投稿したり、他人の投稿にコメントを残したりすることができます。また、フラッシュカード形式の学習機能を通じて、効率的に単語を記憶するためのツールとしての役割も果たします。

## 設計思想
「学習を孤独な作業にしない」という思想のもと、共有とフィードバックのサイクルを回すことで学習のモチベーションを維持することを目指しています。シンプルで軽量なSingle Page Application (SPA) 構成を採用し、直感的な操作性を重視しています。

## ⚠️ 重要な注意事項
- **ベータ版**: 本アプリケーションは現在ベータ版であり、重要なデータの保存や実運用には推奨されません。
- **制作者について**: 制作者はプログラミングの知識が乏しい個人であり、バグやセキュリティの不備が含まれている可能性があります。
- **AI生成**: このアプリケーションのソースコード、ドキュメント、および設計の大部分は、AI（人工知能）を使用して生成されたものです。
